# Orbit Backup Plugin

A powerful and lightweight WordPress backup plugin that allows you to securely back up your entire website, including themes, plugins, uploads, database, and core files. Designed for reliability and performance, this plugin ensures your website data is always safe and recoverable.

Whether you're a developer, agency, or site owner, this plugin provides complete control over manual and automated backups with an easy-to-use interface.

---

## Features

### Full Website Backup
Backup all critical components of your WordPress installation:

- **Themes** – Safely back up all installed themes
- **Plugins** – Backup all active and inactive plugins
- **Uploads** – Backup media files including images, videos, documents, and other uploads
- **Database** – Complete backup of all WordPress database tables
- **Core Files** – Backup essential WordPress core files for full site recovery

This ensures you can fully restore your website to a working state at any time.

---

### Scheduled Backups (Automated Backups)

Set up automatic backups so you never have to worry about forgetting:

- Schedule backups at regular intervals
- Fully automated process
- Runs silently in the background
- Ensures continuous protection of your website

Perfect for production websites that require regular data protection.

---

### Manual Backup Option

Create backups instantly whenever needed:

- One-click backup functionality
- Ideal before updates, migrations, or major changes
- Gives you full control over backup timing

---

### Safe and Reliable

- Secure backup generation
- Organized backup structure
- Minimal performance impact
- Built using WordPress best practices

---

## What Gets Backed Up

This plugin creates a complete backup of:

- `/wp-content/themes/`
- `/wp-content/plugins/`
- `/wp-content/uploads/`
- WordPress database
- WordPress core files
- Configuration files (including `wp-config.php`)

## Installation

1. Download or clone this repository
2. Upload the plugin folder to `/wp-content/plugins/`
3. Go to WordPress Admin → Plugins
4. Activate the plugin
5. Configure backup settings

---

## Requirements

- WordPress 5.0 or higher
- PHP 7.4 or higher
- Proper file permissions for backup creation
